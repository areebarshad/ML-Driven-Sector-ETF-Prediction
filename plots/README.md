# Plots Directory

This directory contains all visualisations generated by the ML-Driven Sector ETF Prediction pipeline. Plots are organised by analytical category and cover model evaluation, feature diagnostics, return distribution analysis, and risk-adjusted performance attribution across the sector ETF universe.

---

## Folder Breakdown

| Folder | Description |
|--------|-------------|
| `Feature_Importances/` | Per-sector horizontal bar charts of the top 20 LightGBM feature importances, measured by cumulative split gain across all trees. High-importance features indicate the variables that most reduce the loss function (MSE) during gradient boosting. Sector-specific variation in feature rankings reveals which signal types (momentum, macro regime, cross-sectional correlation) dominate in each GICS sector. |
| `Price_Trends_&_Return_Distributions/` | (1) Time-series plots of non-stationary adjusted closing price levels across all sector ETFs — used to identify structural breaks, momentum regimes, and cross-sectional divergence over the 2008–2025 sample window. (2) Box plots of daily return distributions exposing central tendency, interquartile spread, directional skewness, and excess kurtosis (heavy tails) per sector. |
| `Returns_Correlation_Heatmap/` | Pearson correlation matrix heatmap of daily sector returns. Values close to +1 indicate strong positive co-movement (reduced diversification benefit); values close to 0 indicate orthogonal risk factor exposures. This informs portfolio construction by identifying sectors that provide genuine cross-sectional diversification versus those that are driven by the same underlying economic risk factors. |
| `R²_By_Sector/` | Bar chart of the **mapped** out-of-sample R² across sectors — the regression score of the classifier's probability→return mapping `(P−0.5)·2·vol` against the actual forward return. These values are negative for every sector by construction (the mapping is a magnitude proxy, not a calibrated regressor); the chart documents that honestly. Directional skill is shown in `Classification_Metrics/` instead. |
| `Classification_Metrics/` | Bar charts of the directional classifier's native metrics: **accuracy** and **ROC-AUC** by sector, each annotated with the 0.50 coin-flip / random baseline. These are the primary skill metrics for the hybrid model — accuracy above 0.50 and AUC above 0.50 indicate genuine directional signal. |
| `Sector_Predictions/` | Time-series overlay plots of realised vs model-predicted 5-day forward returns for each sector ETF. These visualisations qualitatively assess directional accuracy, prediction magnitude, and performance degradation during high-volatility regimes (e.g. COVID-19 shock, rate-hike cycles). Each subplot corresponds to a single sector's out-of-fold (OOF) prediction sequence from the walk-forward cross-validation. |
| `Sharpe_Ratio_By_Sector/` | Bar charts of the annualised Sharpe ratio — defined as (μ − r_f) / σ × √252, with r_f = 3% — for each sector ETF computed from historical daily returns. The Sharpe ratio is the canonical risk-adjusted performance measure; values above 1.0 are conventionally considered acceptable, above 2.0 strong, and above 3.0 exceptional. Provides a benchmark for contextualising the model's directional strategy Sharpe against passive sector exposure. |
| `Portfolio_Backtest/` | Cumulative equity curve (growth of \$1, gross of costs) for the cross-sectional long-short strategy built from the walk-forward out-of-fold predictions. Overlays the proportional-weight and top-N long-short schemes, annotated with each scheme's un-leaked walk-forward trading Sharpe. Positions formed at `t` earn the next-day realized return, so the curve reflects strictly out-of-sample P&L. |

## Layout & Naming

Per-sector plots are nested one directory per ticker, e.g.
`Sector_Predictions/XLK/prediction.png` and `Feature_Importances/XLK/feature_importance.png`.
Universe-level charts sit directly in their category folder (e.g.
`Classification_Metrics/accuracy_by_sector.png`, `Portfolio_Backtest/equity_curve.png`).
Sector tickers are the SPDR Select Sector ETF identifiers: XLK, XLF, XLV, XLP, XLI, XLB,
XLU, XLRE, XLY.

The entire `plots/` tree is regenerated on each run (Stage-1 headless `savefig`). This
folder is rebuilt from scratch — stale assets are removed before a fresh run populates it.

---

## Generation

All plots are produced by `LGBM_Prediction_Model/LightGBM_prediction_model.py`, the
consolidated orchestrator, which saves every figure to the folders above during its
Stage-1 headless pass. The per-stage driver scripts (`sector_eda/`,
`sector_predictive_analysis/`, `sector_risk_analysis/`, `portfolio_backtest/`) regenerate
their respective subsets when run individually.
